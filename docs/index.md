# GOAnno4Bioc

<div class="section level1">

This package for R (&gt;= 4.6.0) defines Bioconductor interfaces to Gene
Ontology Annotation resources from GO Consortium.

-   An exemplary gaf.gz file from March 2026 is provided in inst/human

-   Supports parsing of GAF and tidy manipulations of GO annotations to
    model organisms
