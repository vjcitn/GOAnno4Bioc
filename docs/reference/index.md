# Package index

<div class="section level2">

## All functions

</div>

<div class="section level2">

-   `GAF_ASPECT_TO_RO` : Aspect-to-RO predicate map
-   `GAF_COLUMNS` : GAF 2.2 column names in order
-   `GAF_EVIDENCE_COMPUTATIONAL` : Computational evidence codes
-   `GAF_EVIDENCE_EXPERIMENTAL` : Experimental evidence codes
-   `gaf_add_subject_curie()` : Extract the primary database accession
    as a CURIE
-   `gaf_aspect_summary()` : Summarise annotation counts by aspect and
    evidence type
-   `gaf_cache()` : Retrieve and cache a GAF file via BiocFileCache
-   `gaf_evidence_summary()` : Summarise annotation counts by evidence
    code
-   `gaf_expand_column()` : Expand a pipe-separated GAF column into long
    format
-   `gaf_interspecies()` : Filter to interacting-taxon annotations
-   `gaf_known_species()` : List species with known GAF download URLs
-   `gaf_to_term_association()` : Convert a parsed GAF tibble to semsql
    term\_association format
-   `get_gaf()` : Parse a species GAF file in one step
-   `parse_gaf()` : Parse a GAF 2.x annotation file into a tibble
