# GAF 2.2 column names in order

<div class="ref-description section level2">

A character vector of the 17 column names used in GAF 2.2 files, in
their defined order. Used internally by `parse_gaf` to name the columns
of the returned tibble.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
GAF_COLUMNS
```

</div>

</div>

<div class="section level2">

## Format

An object of class `character` of length 17.

</div>
