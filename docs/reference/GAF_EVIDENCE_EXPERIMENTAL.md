# Experimental evidence codes

<div class="ref-description section level2">

GAF evidence codes considered to be based on direct experimental
evidence, as opposed to computational or author-inferred codes. Useful
for filtering to high-confidence annotations.

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
GAF_EVIDENCE_EXPERIMENTAL
```

</div>

</div>

<div class="section level2">

## Format

An object of class `character` of length 11.
